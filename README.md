git init 初始化

git pull 下載更新

git status 顯示狀態

git log 顯示記錄檔

pwd 顯示自己目前的路徑

ls 顯示目前資料夾裡

touch (檔名.副檔名)產生新檔案

git add(檔名.副檔名) 添加檔案

git commit -m "xxxx" 提交

git push 上傳

----------刪除資料夾------------

git rm -r --cached 資料夾名稱 

git commit -m "刪除訊息"

git push 將刪除的動作上傳

-------------------------------

----------上傳資料夾------------

git add 資料夾名稱 上傳資料夾

git commit -m"上傳訊息"

git push 將上傳的動作上傳

-------------------------------
